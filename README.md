# Abynx Studios – Artist Portfolio Website

A modern and responsive portfolio website built to showcase my creative identity, music releases, editorial visuals, and project work.

This project represents my journey as both a creator and a developer, combining design, branding, and frontend development into one professional digital platform.

## Live Demo

🌐 Live Website: https://abynxstudios.vercel.app

## Features

- Responsive design for desktop, tablet, and mobile
- Custom music showcase section
- Editorial visuals and project portfolio
- About and artist journey section
- Contact and collaboration page
- Optimized media assets for better performance

## Tech Stack

- HTML5
- CSS3
- JavaScript
